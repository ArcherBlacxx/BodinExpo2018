Bootstrap 4 source SASS files will be copied to the bootstrap_theme folder in your project.

The main file for your Bootstrap theme - custom.scss - will be created there as well.

These SASS files are compiled into the existing Bootstrap CSS stylesheet that is linked to the page.

You'll be able to customize Bootstrap variables in the Style panel.

Check out the documentation to learn more about customizing Bootstrap with Pinegrow.
